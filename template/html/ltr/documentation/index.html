Vuexy Admin Template Documentation

Thank you 🙏 for purchasing Vuexy admin template.

Created By 🛠: PIXINVENT Team

Vuexy admin is a super flexible, powerful, clean & modern responsive bootstrap admin template with unlimited possibilities. It includes 5 pre-built templates with organized folder structure, clean & commented code, 100+ pages, 70+ components, 50+ charts, 50+ advance cards (widgets) and many more.

Vuexy admin provides searchable navigation, unique menu layouts, advance cards, and incredible support. Vuexy admin can be used for any type of web application: Project Management, eCommerce backends, CRM, Analytics, Fitness, or any custom admin panels.

It comes with 2 niche dashboards. Vuexy admin template is powered with HTML 5, SASS & Twitter Bootstrap 4 which looks great on Desktops, Tablets, and Mobile Devices. Vuexy admin template comes with a starter kit which will help developers to get started quickly.

If you have any queries that are beyond the scope of this help file, please feel free to create a support ticket on our support portal: https://pixinvent.ticksy.com/

Heads Up!

It is recommended to use any local server (i.e XAMPP, WAMP or Node, etc...) with a protocol "http://" to run this template as it uses AJAX technology as a part of its functionality. It may break some functionality if you opened directly from your HDD in a browser with a protocol “file:///...” (like “file:///C:/path/to/some_page.html”).

Tip

Node.js and Gulp are optional to Use or Run Vuexy Admin HTML version. You can still use and customize easily the HTML version without knowledge of Node.js and Gulp.

Please read our FAQs page for common customization and usage related questions and their solutions.

This documentation covers HTML and Laravel both, please refer Laravel documentation here.
